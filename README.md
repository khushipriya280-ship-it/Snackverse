# SnackVerse

SnackVerse is a modern, responsive web application designed to help beginners, home cooks, and mothers prepare trending Gen Z snacks easily at home. The goal is to provide simple, interactive, guided instructions to reduce dependency on outside junk food by making home cooking accessible and fun.

## Key Features

- **Real-Time Recipe Search**: Quickly filter trending recipes by name, description, or categories (such as pizza, burgers, fries, or drinks).
- **Interactive Checklists**: Keep track of ingredients and cooking steps with interactive checkable items to follow recipes step-by-step.
- **Categorized Sections**: Browse popular food and drink categories at a glance.
- **Responsive Layout**: Designed for seamless viewing across smartphones, tablets, and desktop computers.
- **Modern UI Elements**: Leverages Lucide Icons and customized CSS styles for an aesthetic, clean user experience.

## Recipes Included

- **Homemade Cheesy Pizza**: Crispy, restaurant-style cheese pizza made with simple ingredients.
- **Crispy Pesto Eggs**: Sunny-side up eggs cooked in flavorful, aromatic basil pesto.
- **Baked Feta Pasta**: Rich, creamy oven-baked feta cheese and sweet cherry tomato pasta.
- **Beef Birria Tacos**: Slow-cooked beef in savory consomme, folded in crispy cheese tacos.
- **Golden Sweet Croffle**: Flaky croissant dough pressed into a waffle maker, sweet and crispy.
- **Dalgona Coffee**: Frothy whipped coffee foam served over iced fresh creamy milk.
- **Strawberry Tanghulu**: Crunchy, glass-like sugar shell-coated sweet strawberries.
- **Kewpie Ramen Hack**: Creamy, rich garlic-infused egg and mayo upgraded instant ramen.

## Technology Stack

- **HTML5**: Structured semantic pages.
- **CSS3**: Modern, custom stylesheets with smooth animations, grid layouts, and custom properties.
- **JavaScript (Vanilla)**: Search filtering, clear-query logic, and interactive components.
- **Lucide Icons**: Clean, developer-friendly iconography library.

## Directory Structure

The project codebase is organized into structured subdirectories:
- `/` - Main homepage (`index.html`), search page (`searched.html`), and collection hub pages (`pizza.html`, `burger.html`, `fries.html`, `midnight-cravings.html`).
- `/css/` - Custom stylesheets (`style.css`).
- `/js/` - JavaScript logic files (`script.js`, `global-search.js`).
- `/images/` - Category banner and recipe image assets.
- `/recipes/` - 48 individual recipe detail pages.

## Getting Started

Since this is a client-side static web application, no compilation or local server setup is required.

### Prerequisites

You only need a modern web browser (such as Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari) to run the application.

### Running Locally

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/khushipriya280-ship-it/Snackverse.git
   ```

2. Navigate into the project directory:
   ```bash
   cd Snackverse
   ```

3. Open `index.html` in your web browser. You can do this by double-clicking the file or right-clicking and selecting your preferred browser.
